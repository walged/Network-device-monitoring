# Icons

Поместите сюда иконки приложения:

- `icon.ico` - иконка для Windows (256x256)
- `icon.png` - иконка PNG (512x512)
- `tray-icon.png` - иконка для трея (16x16 или 32x32)
- `app-icon.png` - иконка приложения (512x512)

Можете использовать сервисы для генерации иконок:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
- https://www.icoconverter.com/
